# SK8HUB E-commerce

Um mini e-commerce/landing page moderno focado em produtos de skate e cultura streetwear.

## Descrição

Este projeto apresenta uma loja online premium para produtos de skate, incluindo shapes, rodas, trucks, tênis, roupas streetwear e acessórios. O design é moderno, urbano e visualmente impressionante, transmitindo energia, movimento e estilo street.

## Funcionalidades

- **Hero Section**: Imagem cinematográfica com overlay e chamada para ação
- **Produtos**: Grid responsivo com filtros por categoria
- **Carrinho Simulado**: Contador de itens no carrinho
- **Seções Informativas**: Benefícios, promoções e galeria
- **Responsividade Completa**: Desktop, tablet e mobile
- **Animações Suaves**: Fade-in, hover effects e transições
- **Menu Mobile**: Navegação otimizada para dispositivos móveis

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com variáveis CSS, Flexbox e Grid
- **JavaScript (Vanilla)**: Funcionalidades interativas
- **JSON**: Armazenamento de dados dos produtos
- **Font Awesome**: Ícones
- **Google Fonts**: Tipografia Montserrat

## Como Executar

1. Clone ou baixe o projeto
2. Abra o terminal na pasta do projeto
3. Execute um servidor local:
   ```bash
   python -m http.server 8000
   ```
4. Abra o navegador e acesse `http://localhost:8000`
